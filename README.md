Numbers Converter
=================

A simple number conversion program.

Compiling
---------
### On GUN/Linux

#### Dependencies

| Dependency |  Version  |
|------------|-----------|
| GCC        | 4.8+      |
| CMake      | 3.12+     |

#### Debian/Ubuntu:
	sudo apt install libgtk-3-dev cmake make gcc

#### Build
	cmake .
	make
	./NumbersConverter
